  • Créer une image de construction
  • Dépendances requises :
    • unzip
    • build-essential
    • autoconf
    • bison
ul>li*3

Construire PHP

Version light, sans aucune extension


FROM debian:jessie
MAINTAINER Pascal MARTIN <contact@pascal-martin.fr>

RUN apt-get update && \
    apt-get install -y unzip ↵
        build-essential autoconf bison && \
    apt-get clean

COPY ./files/make-php.sh /tmp/
ul>li*3

Construire PHP

  • Récupérer les sources
  • buildconf
  • configure
    • --disable-all
    • --prefix=...
  • make
  • make install
ul>li*3

Construire PHP


cd /tmp/
unzip php-src-master.zip

cd /tmp/php-src-master/
./buildconf
./configure --disable-all ↵
    --prefix=/usr/local/php-7.1-dev
make -j 2 -l 4
make install
                        

/usr/local/php-7.1-dev/bin/php --version
/usr/local/php-7.1-dev/bin/php -m
